# Paradise Nursery Shopping Application

## Project Name
**Paradise Nursery – Online Plant Shop**

## Project Description
Paradise Nursery is a dynamic, fully interactive e-commerce web application built for an online plant shop. Users can browse a curated collection of plants, view detailed information including images, names, descriptions, and prices, add items to a shopping cart, and manage their cart in real time.

## Features
- 🌿 **Landing Page** – Welcoming hero section with the shop name and a call-to-action
- 🧭 **Navigation Bar** – Links to Home and Plants sections, plus a live cart icon
- 🛒 **Shopping Cart** – Slide-out basket with item management and total cost calculation
- ➕ **Add / Remove Items** – Increase, decrease, or delete items directly from the cart or product card
- 💰 **Dynamic Pricing** – Total cost updates automatically as quantities change
- 🌱 **Plant Categories** – Air Purifying, Tropical, and Succulents & Cacti

## Project Structure

paradise-nursery/
